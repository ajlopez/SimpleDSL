# SimpleDSL

Simple DSL (Domain-Specific Language) helper implementation, in JavaScript/Node.js.

## Installation

Via npm on Node:

```
npm install simpledsl
```


## Usage

Reference in your program:

```js
var sdsl = require('simpledsl');
```
